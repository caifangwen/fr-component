总览与使用建议

HTML 目前 Living Standard 中定义了上百个标签，本文档精选 106 个最常用、最核心的标签进行系统梳理。掌握这些标签足以覆盖 99% 的日常开发场景。

选标签的黄金法则
  • 语义优先：有专门标签时绝不用 div/span 代替（如用 <article> 而非 <div class="article">）
  • 可访问性：img 必写 alt，form 控件必关联 label，按钮必用 <button> 而非 div
  • SEO 友好：h1-h6 构成文档大纲，time 标注日期，address 放联系信息
  • 渐进增强：details/summary、dialog 等原生交互标签能减少大量 JS

标签类型速查

类型标识含义特征
块级Block独占一行，可设置宽高，如 div、p、section
内联Inline不独占一行，跟随文本流，如 span、a、strong
空标签Void无闭合标签，如 img、br、input、meta
元数据Meta不可见或脚本相关，如 head、script、template

1. 文档根与元数据 8 个

<html> 块级

属性：lang, xmlns

HTML 文档的根元素，所有其他元素都是其后代。通过 lang 属性声明页面语言，对屏幕阅读器和 SEO 都至关重要。

<html lang="zh-CN">
  ...
</html>

<head> 元数据

属性：无全局唯一属性

包含文档的元数据集合，如标题、字符集声明、样式表链接、脚本等。内容不会直接渲染在页面中。

<head>
  <meta charset="UTF-8">
  <title>页面标题</title>
</head>

<body> 块级

属性：全局事件属性（如 onload）

文档的主体，包含所有可见内容，如文本、图片、视频、游戏等。

<body>
  <h1>Hello World</h1>
</body>

<title> 元数据

属性：无

定义文档标题，显示在浏览器标签页和收藏夹中，对 SEO 极为重要。

<title>页面标题</title>

<meta> 空标签

属性：charset, name, content, http-equiv

表示那些不能被 title、base、link、style、script 表达的元数据，如字符集、视口、描述。

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<link> 空标签

属性：href, rel, type, media, crossorigin

链接外部资源，最常用的是引入 CSS 样式表，也可用于设置网站图标（favicon）。

<link rel="stylesheet" href="style.css">
<link rel="icon" href="favicon.ico">

<base> 元数据

属性：href, target

设置文档中所有相对 URL 的基准地址，以及链接的默认打开目标。

<base href="https://example.com/" target="_blank">

<style> 元数据

属性：media, type

包含文档的内联 CSS 样式信息，可放在 head 或 body 中（推荐 head）。

<style>
  body { background: #fff; }
</style>

2. 脚本与模板 3 个

<script> 元数据

属性：src, type, async, defer, crossorigin, integrity

嵌入或引用可执行脚本（通常是 JavaScript）。可放在 head 或 body 末尾以优化加载。

<script src="app.js" defer></script>
<script>console.log('hi');</script>

<noscript> 块级

属性：全局属性

当浏览器禁用脚本或浏览器不支持脚本时显示的内容。常用于提示用户启用 JavaScript。

<noscript>
  <p>请启用 JavaScript 以获得最佳体验。</p>
</noscript>

<template> 模板

属性：全局属性

保存 HTML 模板片段，页面加载时不会渲染，可通过 JavaScript 克隆后插入文档。常用于 Web 组件。

<template id="card-tpl">
  <div class="card"></div>
</template>

3. 区块与标题 15 个

<h1> – <h6> 块级

属性：全局属性

六级标题，h1 最高，h6 最低。构成文档大纲，对 SEO 和可访问性至关重要。一个页面建议只用一个 h1。

<h1>一级标题</h1>
<h2>二级标题</h2>

<header> 块级

属性：全局属性

表示页面或区块的头部，通常包含导航、logo、搜索框等。一个页面可有多个 header。

<header>
  <nav>...</nav>
</header>

<nav> 块级

属性：全局属性

定义导航链接区域，放置主要导航链接。并非所有链接都必须放在 nav 中。

<nav>
  <a href="/">首页</a>
  <a href="/about">关于</a>
</nav>

<main> 块级

属性：全局属性

文档的主要内容区域，每个页面只能有一个 main，且不能是 article、aside、footer、header、nav 的后代。

<main>
  <article>...</article>
</main>

<article> 块级

属性：全局属性

表示文档中独立的、可独立分发或复用的内容，如博客文章、新闻、论坛帖子。

<article>
  <h2>文章标题</h2>
  <p>文章内容...</p>
</article>

<section> 块级

属性：全局属性

表示文档中的通用独立区块，通常带有标题。比 div 更具语义，但比 article 独立性弱。

<section>
  <h2>章节标题</h2>
  <p>内容...</p>
</section>

<aside> 块级

属性：全局属性

表示与周围内容间接相关的部分，如侧边栏、广告、引用块、相关链接列表。

<aside>
  <h3>相关文章</h3>
  <ul>...</ul>
</aside>

<footer> 块级

属性：全局属性

表示页面或区块的底部，通常包含版权信息、作者信息、相关文档链接。

<footer>
  <p>© 2026 公司名</p>
</footer>

<div> 块级

属性：全局属性

通用容器，无语义，仅用于布局和样式。当没有其他语义标签适用时使用。

<div class="wrapper">
  <div class="item">...</div>
</div>

<address> 块级

属性：全局属性

提供某人、某组织的联系信息。通常包含在 footer 或 article 中。不能用于任意地址文本。

<address>
  <a href="mailto:hi@example.com">hi@example.com</a>
</address>

4. 文本内容 12 个

<p> 块级

属性：全局属性

表示一个文本段落。浏览器会在相邻 p 元素之间自动添加垂直间距。

<p>这是一个段落。</p>
<p>这是另一个段落。</p>

<hr> 空标签

属性：全局属性

水平分割线，表示段落级元素之间的主题转换（thematic break）。

<p>第一部分</p>
<hr>
<p>第二部分</p>

<pre> 块级

属性：全局属性

预格式化文本，保留空格和换行符。常用于显示代码块或 ASCII 艺术。

<pre>
  第一行
  第二行
</pre>

<blockquote> 块级

属性：cite

块级引用，表示引用的内容。可通过 cite 属性给出来源 URL。

<blockquote cite="https://example.com">
  <p>引用的内容...</p>
</blockquote>

<ol> 块级

属性：reversed, start, type

有序列表，列表项通常带有数字或字母序号。

<ol start="3">
  <li>第三项</li>
  <li>第四项</li>
</ol>

<ul> 块级

属性：全局属性

无序列表，列表项通常带有项目符号（圆点、方块等）。

<ul>
  <li>苹果</li>
  <li>香蕉</li>
</ul>

<li> 块级

属性：value（仅用于 ol 子项）

列表项，必须嵌套在 ol、ul 或 menu 中。可以包含其他块级元素。

<li>列表项内容</li>

<dl> 块级

属性：全局属性

描述列表（definition list），用于展示键值对、术语与解释等成对信息。

<dl>
  <dt>术语</dt>
  <dd>术语的解释</dd>
</dl>

<dt> 块级

属性：全局属性

描述列表中的术语（description term），与 dd 配合使用。

<dt>HTML</dt>

<dd> 块级

属性：全局属性

描述列表中的定义或解释（description definition），与 dt 配合使用。

<dd>超文本标记语言</dd>

<figure> 块级

属性：全局属性

自包含的内容块，如图表、插图、代码片段等，通常与 figcaption 一起使用。

<figure>
  <img src="chart.png" alt="图表">
  <figcaption>图1：销售趋势</figcaption>
</figure>

<figcaption> 块级

属性：全局属性

figure 元素的标题或说明文字，必须作为 figure 的第一个或最后一个子元素。

<figcaption>图1：销售趋势</figcaption>

5. 内联文本语义 28 个

<a> 内联

属性：href, target, rel, download, hreflang

锚点元素，创建指向其他页面、文件、邮箱地址或同一页面内部位置的超链接。

<a href="https://example.com" target="_blank">访问网站</a>

<em> 内联

属性：全局属性

表示强调，通常渲染为斜体。改变句子的语义重音。

<p>你<em>必须</em>完成这个任务。</p>

<strong> 内联

属性：全局属性

表示强烈的重要性、严重性或紧迫性，通常渲染为粗体。

<p><strong>警告：</strong>请勿触摸。</p>

<small> 内联

属性：全局属性

表示边注、免责声明、版权声明等小号文本，语义上表示次要内容。

<p><small>© 2026 版权所有</small></p>

<s> 内联

属性：全局属性

表示不再准确或不再相关的内容，如原价划线。不要与 del 混淆。

<p>原价 <s>¥199</s>，现价 ¥99</p>

<cite> 内联

属性：全局属性

标记作品的标题（书籍、歌曲、电影、论文等），通常渲染为斜体。

<p>推荐阅读 <cite>HTML 权威指南</cite></p>

<q> 内联

属性：cite

短行内引用，浏览器通常会自动添加引号。可通过 cite 属性给出来源。

<p>他说 <q>时间就是金钱</q></p>

<dfn> 内联

属性：title（定义术语）

表示正在定义的术语，其父级段落、section 应包含该术语的定义。

<p><dfn>HTML</dfn> 是超文本标记语言。</p>

<abbr> 内联

属性：title

表示缩写或首字母缩略词。title 属性提供完整展开形式。

<abbr title="HyperText Markup Language">HTML</abbr>

<ruby> 内联

属性：全局属性

用于展示东亚文字的注音（ruby 注释），如汉字旁标注拼音或假名。

<ruby>汉 <rt>han</rt></ruby>

<rt> 内联

属性：全局属性

ruby 注音文本，放置在 ruby 元素内，提供主文本的发音注释。

<rt>ふりがな</rt>

<rp> 内联

属性：全局属性

在浏览器不支持 ruby 时显示括号或其他后备内容，包裹 rt 元素。

<ruby>漢 <rp>(</rp><rt>かん</rt><rp>)</rp></ruby>

<data> 内联

属性：value

将内容与机器可读的翻译关联，通过 value 属性提供对应的值。

<data value="398">新品</data>

<time> 内联

属性：datetime

表示特定的时间段或日期，datetime 属性提供机器可读格式，利于 SEO。

<time datetime="2026-05-08">2026年5月8日</time>

<code> 内联

属性：全局属性

表示计算机代码片段，等宽字体显示。适合行内代码引用。

<p>使用 <code>console.log()</code> 输出。</p>

<var> 内联

属性：全局属性

表示数学表达式或编程上下文中的变量名，通常斜体显示。

<p>面积 = <var>r</var>² × π</p>

<samp> 内联

属性：全局属性

表示计算机程序输出样本，等宽字体显示。

<p>系统提示：<samp>操作成功</samp></p>

<kbd> 内联

属性：全局属性

表示键盘输入或用户需要按下的按键，常用于快捷键说明。

<p>按 <kbd>Ctrl</kbd> + <kbd>S</kbd> 保存</p>

<sub> 内联

属性：全局属性

下标文本，用于化学式、数学下标等场景。

<p>H<sub>2</sub>O 是水。</p>

<sup> 内联

属性：全局属性

上标文本，用于数学指数、脚注标记等场景。

<p>E = mc<sup>2</sup></p>

<i> 内联

属性：全局属性

表示与周围文本不同的语气或质量，如外文短语、技术术语、想法。默认斜体。

<p><i>caveat emptor</i>（买者自负）</p>

<b> 内联

属性：全局属性

引起注意但不增加重要性，如摘要关键词、评论中的产品名。默认粗体。

<p><b>注意</b>：这是关键步骤。</p>

<u> 内联

属性：全局属性

表示非文本注释，如中文专有名词需要标注、拼写检查错误。默认下划线。

<p><u>北京</u>是中国的首都。</p>

<mark> 内联

属性：全局属性

标记文本中因上下文相关而高亮的部分，如搜索结果高亮。默认黄底。

<p>搜索结果：<mark>关键词</mark></p>

<bdo> 内联

属性：dir（必填）

双向覆盖（Bidirectional Override），强制覆盖当前文本的双向算法方向。

<bdo dir="rtl">这段文字从右向左显示</bdo>

<span> 内联

属性：全局属性

通用行内容器，无语义。用于对行内元素进行样式控制或脚本操作。

<p>价格是 <span class="price">¥99</span></p>

<br> 空标签

属性：全局属性

换行符，在文本中插入一个简单的换行。不要在段落间使用 br 代替 p。

<p>第一行<br>第二行</p>

<wbr> 空标签

属性：全局属性

可选换行机会，指示浏览器在必要时可以在此处换行，常用于长 URL 或单词。

<p>https://example.com/<wbr>very-long-path</p>

6. 图片与多媒体 9 个

<img> 空标签

属性：src, alt, width, height, loading, srcset, sizes

将图像嵌入文档。必须包含 alt 属性以支持可访问性和 SEO。

<img src="photo.jpg" alt="风景照" width="600" loading="lazy">

<audio> 块级

属性：src, controls, autoplay, loop, muted, preload

在文档中嵌入音频内容，支持多种音频格式。可包含 source 子元素。

<audio controls>
  <source src="music.mp3" type="audio/mpeg">
</audio>

<video> 块级

属性：src, controls, autoplay, loop, muted, width, height, poster

在文档中嵌入视频内容。可包含 source、track 等子元素。

<video controls width="640" poster="thumb.jpg">
  <source src="movie.mp4" type="video/mp4">
</video>

<source> 空标签

属性：src, type, media, sizes, srcset

为 video、audio、picture 指定多种媒体资源，浏览器会选择最合适的格式。

<source src="movie.webm" type="video/webm">

<track> 空标签

属性：kind, src, srclang, label, default

为 video、audio 指定外部文本轨道，如字幕、标题、章节等。

<track kind="subtitles" src="subs.vtt" srclang="zh" label="中文字幕">

<canvas> 块级

属性：width, height

通过脚本（通常是 JavaScript）绘制图形的画布，可用于游戏、图表、图像处理。

<canvas id="myCanvas" width="400" height="300"></canvas>

<svg> 内联

属性：xmlns, viewBox, width, height, fill, stroke

定义可缩放矢量图形（SVG），可直接嵌入 HTML，不会失真，适合图标和插图。

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="red" />
</svg>

<map> 内联

属性：name

与 img 配合定义图像映射（可点击区域），包含多个 area 子元素。

<map name="shapes">
  <area shape="rect" coords="34,44,270,350" href="rect.htm">
</map>

<area> 空标签

属性：shape, coords, href, alt, target, download

在图像映射中定义可点击的热区区域，必须嵌套在 map 元素内。

<area shape="circle" coords="75,75,75" href="circle.htm" alt="圆形">

7. 嵌入内容 5 个

<iframe> 块级

属性：src, width, height, name, sandbox, allow, loading

内联框架，在当前页面中嵌入另一个 HTML 页面。常用于嵌入地图、广告、第三方组件。

<iframe src="map.html" width="600" height="400"></iframe>

<embed> 空标签

属性：src, type, width, height

嵌入外部内容，如 Flash（已淘汰）、PDF 阅读器。推荐使用 iframe 或 object 替代。

<embed src="file.pdf" type="application/pdf" width="600" height="400">

<object> 块级

属性：data, type, width, height, name, form

通用外部资源嵌入容器，可嵌入图片、视频、PDF、Flash 等。支持包含 param 配置参数。

<object data="movie.swf" type="application/x-shockwave-flash">
  <param name="quality" value="high">
</object>

<param> 空标签

属性：name, value

为 object 元素定义参数，如初始化配置、播放设置等。

<param name="autoplay" value="true">

<picture> 块级

属性：全局属性

响应式图片容器，包含多个 source 和一个 img，根据媒体查询选择最佳图片。

<picture>
  <source srcset="img.webp" type="image/webp">
  <img src="img.jpg" alt="描述">
</picture>

8. 表格 10 个

<table> 块级

属性：全局属性

表格容器，包含行和单元格的二维数据表。不应被滥用于页面布局。

<table>
  <tr><th>名称</th><th>价格</th></tr>
  <tr><td>苹果</td><td>5</td></tr>
</table>

<caption> 块级

属性：全局属性

表格标题，必须作为 table 的第一个子元素。描述表格的整体内容。

<caption>2026年销售数据</caption>

<colgroup> 块级

属性：span

定义表格中的一组列，用于统一设置样式。包含一个或多个 col 元素。

<colgroup>
  <col style="background:#f0f0f0">
  <col span="2">
</colgroup>

<col> 空标签

属性：span

定义表格中一列或多列的属性，必须嵌套在 colgroup 中。

<col span="2" style="width:150px">

<tbody> 块级

属性：全局属性

表格主体，包含数据行。一个 table 可包含多个 tbody 以对数据进行逻辑分组。

<tbody>
  <tr><td>数据1</td></tr>
</tbody>

<thead> 块级

属性：全局属性

表格头部，通常包含列标题行（th）。打印时可在每页重复显示。

<thead>
  <tr><th>月份</th><th>收入</th></tr>
</thead>

<tfoot> 块级

属性：全局属性

表格页脚，通常包含汇总行（如总计、平均值）。必须出现在 tbody 之前（DOM 顺序）。

<tfoot>
  <tr><td>总计</td><td>1000</td></tr>
</tfoot>

<tr> 块级

属性：全局属性

表格行（table row），包含 td 或 th 单元格。

<tr>
  <td>单元格1</td>
  <td>单元格2</td>
</tr>

<td> 块级

属性：colspan, rowspan, headers

表格数据单元格（table data），包含具体数据内容。

<td colspan="2">合并两列</td>

<th> 块级

属性：colspan, rowspan, headers, scope, abbr

表头单元格（table header），通常加粗居中显示，表示列或行的标题。

<th scope="col">产品名称</th>

9. 表单 14 个

<form> 块级

属性：action, method, enctype, name, target, novalidate

表单容器，包含交互式控件，用于向服务器提交用户数据。

<form action="/submit" method="POST">
  ...
</form>

<label> 内联

属性：for（关联控件 id）, form

为表单控件定义说明标签。点击 label 可聚焦关联的控件，极大提升可访问性。

<label for="email">邮箱：</label>
<input id="email" type="email">

<input> 空标签

属性：type, name, value, placeholder, required, disabled, readonly

通用的表单输入控件，根据 type 属性呈现为文本框、密码框、复选框、单选按钮、日期选择器等。

<input type="text" name="user" placeholder="用户名" required>

<button> 内联

属性：type, name, value, disabled, form

可点击的按钮，比 input type="button" 更灵活，可包含 HTML 内容（图标、样式文本等）。

<button type="submit">提交订单</button>

<select> 内联

属性：name, size, multiple, required, disabled

选项菜单（下拉列表），包含 option 和 optgroup 子元素。支持多选（multiple）。

<select name="city">
  <option value="bj">北京</option>
  <option value="sh">上海</option>
</select>

<datalist> 块级

属性：id（被 input list 属性引用）

为 input 提供预定义选项列表，用户可输入或从建议中选择，兼具灵活性和便利性。

<input list="browsers">
<datalist id="browsers">
  <option value="Chrome">
  <option value="Firefox">
</datalist>

<optgroup> 块级

属性：label, disabled

在 select 中对 option 进行分组，通过 label 属性显示分组名称，不可选择。

<optgroup label="水果">
  <option>苹果</option>
  <option>香蕉</option>
</optgroup>

<option> 块级

属性：value, selected, disabled, label

select、optgroup 或 datalist 中的选项。可设置 value 和默认选中（selected）。

<option value="1" selected>选项一</option>

<textarea> 内联

属性：name, rows, cols, maxlength, placeholder, readonly

多行纯文本编辑控件，适合评论、留言等长文本输入。

<textarea name="msg" rows="4" cols="50">默认内容</textarea>

<output> 内联

属性：for, form, name

表示计算或用户操作的结果，常用于实时显示表单计算值。通过 for 关联输入源。

<output for="a b" name="sum">0</output>

<progress> 内联

属性：value, max

显示任务完成进度条。通过 value 和 max 表示当前进度。

<progress value="70" max="100">70%</progress>

<meter> 内联

属性：value, min, max, low, high, optimum

表示已知范围内的标量值或分数值，如磁盘使用率、评分。不同于 progress，它表示静态测量值。

<meter value="0.6">60%</meter>

<fieldset> 块级

属性：disabled, form, name

将表单中的相关控件分组，通常与 legend 配合使用。可整体禁用（disabled）。

<fieldset>
  <legend>个人信息</legend>
  ...
</fieldset>

<legend> 块级

属性：全局属性

为 fieldset 定义标题说明，必须作为 fieldset 的第一个子元素。

<legend>配送地址</legend>

10. 交互元素 2 个

<details> 块级

属性：open（默认展开）

创建一个可展开/折叠的详情小部件，用户点击可查看或隐藏附加信息。无需 JavaScript。

<details>
  <summary>更多信息</summary>
  <p>这里是隐藏的详细内容。</p>
</details>

<summary> 块级

属性：全局属性

details 元素的摘要标题，始终可见。点击可切换 details 的展开状态。

<summary>点击展开</summary>
